The Oce vignette provides a good starting point. To get it, install Oce and then type vignette("oce") in an R console.

Dan Kelley and Clark Richards are preparing to write a paper about the use of Oce, although the journal has not yet been selected. The AGU EOS Transactions and the CMOS Bulletin both seem like good choices but both require subscriptions, which defeats the purpose of reaching a wide audience.
