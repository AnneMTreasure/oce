The stable version

The stable version of oce is provided on the R archives. It is installed from within R with {% highlight R %} install.packages("oce") {% endhighlight %}

Pre-compiled development version (for casual users)

About once per week, oce is compiled for MSWindows and for OS X. The results are stored at a Dropbox site. Visit that site and the appropriate subdirectory (osx or windows), and download both oce and ocedata. Then install as, for example on OS X, {% highlight %} R CMD install oce_0.9-18.tgz R CMD install ocedata_0.1.4.tgz {% endhighlight %}

Source code for the development version (for advanced users)

Users who are set up to build packages (which requires a C compiler, etc; see note 1) may stay at the cutting edge by installing a development version. This may be done in several ways, of which the third is probably the best for most users.

  1. Download the source for oce and ocedata as tarballs (oce and ocedata), expand the files, rename the resultant directories to oce and ocedata and then execute the OS commands stated below. {% highlight bash %} R CMD build oce R CMD install oce_0.9-18.tar.gz # adjust version as needed R CMD build ocedata R CMD install ocedata_0.1.4.gz # adjust version as needed {% endhighlight %} A similar procedure works for the zip files at oce and ocedata.
  2. Acquire the sources through Git using {% highlight bash %} git clone https://dankelley@github.com/dankelley/oce.git git clone https://dankelley@github.com/dankelley/ocedata.git {% endhighlight %} and then build them as described in #1.
  3. Download and build using install_github() {% highlight R %} library(devtools) install_github("dankelley/oce", ref="develop") install_github("dankelley/ocedata", ref="master") {% endhighlight %}

Note 1. Getting ready to compile (by installing the gcc compiler, etc.) is easy on linux, although the method depends on the system. On MSWindows, install Rtools from CRAN. OSX instructions are provided at the ATT site ... although it is possible that RStudio comes with the requisite tools, so you might want to try that first.
