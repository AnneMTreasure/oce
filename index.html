Oce is a package in the R statistical language that helps Oceanographers do their work. Dan Kelley, the author of oce, is working on a book entitled Oceanographic Analysis with R that illustrates the use of oce for a wide variety of tasks.

The stable version of oce is provided on the R archives. It is installed from within R, just like other packages. A development version is available as a tarball, as a zipball, and as a git repository (git clone https://dankelley@github.com/dankelley/oce.git ); any of these is built and installed by typing for example 

R CMD build oce
R CMD install oce_0.8-5.tar.gz # change version number as appropriate

at the unix commandline (where the version number and file suffix match the downloaded file).
