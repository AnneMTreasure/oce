Oce is a package in the R statistical language that helps Oceanographers do their work. Dan Kelley, the author of oce, is working on a book entitled Oceanographic Analysis with R that illustrates the use of oce for a wide variety of tasks.

The stable version of oce is provided on the R archives. It is installed from within R, just like other packages. A development version is available as a tarball, as a zipball, and as a git repository, which may be accessed as follows.

git clone https://dankelley@github.com/dankelley/oce.git

This source code may be built and installed by typing the following in a unix shell, from the level above that in which the cloned directory 'oce' exists.

R CMD build oce
R CMD install oce_0.8-8.tar.gz

(It will be necessary to alter version number and file suffix to match the downloaded file).
