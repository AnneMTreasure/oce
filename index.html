About Oce

Oce is a package in the R statistical language that is designed to help Oceanographers do their work. Prior to 2010-03-19, oce was hosted on google-code, where it was called r-oce.

Using oce

The package is normally installed from within R. To report errors in oce, or to suggest features, open an issue.

Developers

To browse the source, visit http://github.com/dankelley/oce. To get a local copy of the source, type 

git clone git@github.com:dankelley/oce.git
in a unix-like shell.
