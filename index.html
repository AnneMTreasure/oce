Oce is a package in the R statistical language that helps Oceanographers do their work. The stable version is provided on the R archives, and is normally installed from within R. The gallery below gives a taste of Oce.

Adventurous users may install the cutting-edge version from the present site, as a tarball, zipball, or via git (git clone git@github.com:dankelley/oce.git the first time, later git pull for updates). Once the source is unpacked and renamed oce, use conventional R commands to build and install the new version: 

R CMD build oce
R CMD install oce_0.1-81.tar.gz # change version number as appropriate

in a unix console. The author is keen to hear about errors, and suggestions for features; to help with that, please open an issue.

(Prior to 2010-03-19, oce was hosted on google-code, where it was called r-oce.)
