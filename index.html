About Oce

Oce is a package in the R statistical language that helps Oceanographers do their work. The stable version is provided on the R archives, and is normally installed from within R. Adventurous users may install the cutting-edge version by typing 

git clone git@github.com:dankelley/oce.git  # later, just do 'git pull'
R CMD build oce
R CMD install oce_0.1-81.tar.gz # change version number as appropriate

in a unix console. The author is keen to hear about errors, and suggestions for features; to help with that, please open an issue.
