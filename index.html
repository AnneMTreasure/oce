The stable version of oce is provided on the R archives, and is installed from within R, just like other packages. A development version is available as a tarball, as a zipball, and as a git repository (git clone https://dankelley@github.com/dankelley/oce.git); any of these is built and installed by typing 

R CMD build oce
R CMD install oce_0.2-1.tar.gz # change version number as appropriate

at the commandline.

Please open an issue to report a bug or suggest a feature.

PS: prior to 2010-03-19, oce was hosted on google-code, where it was called r-oce.
