Oce is a package in the R statistical language that is designed to help Oceanographers do their work. Prior to 2010-03-19, oce was hosted on google-code, where it was called r-oce.

To report errors in oce, or to suggest features, open an issue.

To get a local copy of the up-to-date source, just clone it (e.g. execute git clone git@github.com:dankelley/oce.git in a unix shell).
